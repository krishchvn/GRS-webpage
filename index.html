Introduction

Operation Performed using Gestures

Communication is one of the most important things when it comes to living in a society. Not being able to communicate due to disability in speaking and hearing brings different kinds of challenges to this society.

This project mainly focuses on removing the barrier of communication between the mute community and the people not familiar with concept of sign language.

The purpose of this project is to build a real-time hand gesture recognition system using concepts of Deep Learning and Computer Vision which recognizes gestures and performs operations provided.
